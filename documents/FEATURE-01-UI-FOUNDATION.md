# Feature 01: UI Foundation

## Goal
Create the base UI components with paper-like, colorful, kid-friendly design.

## Tasks

### 1.1 Global Styles
- [ ] Update globals.css with kid-friendly fonts
- [ ] Add paper texture background style
- [ ] Define color palette CSS variables
- [ ] Import a playful font (e.g., Nunito, Comic Neue, Fredoka One)

### 1.2 PaperBackground Component
- [ ] Create paper texture background (cream/beige)
- [ ] Optional: subtle paper grain effect
- [ ] Full viewport coverage

### 1.3 Button Component
- [ ] Large, rounded buttons
- [ ] Multiple color variants (blue, green, orange, pink, purple)
- [ ] Hover/press animations (scale, shadow)
- [ ] Click sound integration (prepared, not implemented yet)

### 1.4 ConfirmDialog Component
- [ ] Modal overlay
- [ ] Friendly message display
- [ ] Yes/No buttons with icons
- [ ] Animation on open/close

### 1.5 BackButton Component
- [ ] Position: top-left
- [ ] Icon: arrow or home
- [ ] Triggers confirmation before leaving game

## Acceptance Criteria
- All components render with consistent style
- Buttons have satisfying visual feedback
- Paper background feels warm and inviting
- Text is large and readable for children
